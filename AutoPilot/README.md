# AutoPilot

## What's this?

Auto pilot to a planet or star system.

This mod is a CruiseAssist extension.

Original mod: [AutoPilot](https://thunderstore.io/package/tanu/AutoPilot/)  
It was discontinued, so I made a fork of it.
